# Contributing

## Commit message (Conventional Commits)

Dùng format:

- `feat: ...` tính năng mới
- `fix: ...` sửa lỗi
- `docs: ...` tài liệu
- `refactor: ...` tái cấu trúc không đổi hành vi
- `chore: ...` việc lặt vặt (tooling, deps)

Ví dụ:

- `feat(web): add Supabase auth pages`
- `fix(web): avoid invalid uuid query when no orders`

## Branch

- `main`: ổn định
- `feature/*`: phát triển tính năng

## Pull request

- Mô tả ngắn gọn mục tiêu
- Liệt kê file chính đã sửa
- Chụp ảnh màn hình nếu thay đổi UI
